# IPL Data Analysis using Python

## Objective

Analyze IPL match and player data to identify top scorers, strike rates, team win percentages, and season-wise performance trends.

## Dataset

The dataset used in this project is the IPL Complete Dataset available on Kaggle.

Source:
https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020

Files used:
- matches.csv
- deliveries.csv
- 
## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Analyses

* Top 10 Run Scorers
* Strike Rate Analysis
* Team Win Percentage
* Orange Cap Winners by Season
* Virat Kohli Performance Across Seasons
* Player Performance Comparison

## Key Insights

* Virat Kohli emerged as one of the highest run scorers in IPL history.
* Teams with consistent performance maintained higher win percentages.
* Strike rates varied significantly among top batsmen.
* Orange Cap winners consistently outperformed league averages.
* Player performance showed noticeable variation across seasons.

## Project Structure

data/ - Raw IPL datasets

visuals/ - Charts and visualizations

IPL_Analysis.ipynb - Complete analysis notebook

Project_Insights.md - Summary of findings
